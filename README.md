# Terraform


### How to use Terraform 

 az login                         ---\n 
 terraform -help                  ---帮助\n
 terraform fmt                    ---自动识别格式\n
 terraform init                   ---初始化\n
 terraform plan                   ---查看代码变更\n
 terraform apply -auto-approve    ---应用代码\n
 terraform destroy                ---摧毁代码\n
